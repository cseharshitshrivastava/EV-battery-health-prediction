# 🔋 Battery Health Prediction System

## 📌 Overview
This project predicts the health of an Electric Vehicle battery using Machine Learning.

## 🚀 Features
- Predict battery capacity
- Simple web interface using Streamlit
- Uses real battery parameters

## 🛠 Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit

## ▶️ How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Train model:
python model.py

3. Run app:
streamlit run app.py

## 📊 Sample Input
Voltage: 3.7  
Current: 1.5  
Temperature: 25  
Cycle: 100  

## 📈 Output
Predicted battery capacity
